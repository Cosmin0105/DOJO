# DOJO

https://github.com/Cosmin0105/DOJO/tree/master



The project is still in the master phase. The website is designed for several dojos in Galați and Brăila, utilizing HTML, CSS, and JavaScript. The site is responsive and easy to use. You can send an email to the administrator through the contact section, view the athletes' matches, and see their activities over the years.
